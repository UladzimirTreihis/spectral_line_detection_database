Submitting Entries from a File

In order to upload multiple detections from a file, you must use a CSV file with correct format. Please download the template file below. Files that do not match the required format be automatically rejected. If your submission is rejected, please see the output warnings for possible corrections, and then resubmit. If you have further issues, please contact the database admin. All submissions will be manually approved before becoming publically accessible.

While entering values, please note that:

  • Coordinate System must be ICRS or J2000 only
  • For the lensing glag, enter either Lensed, Unlensed or Unknown
  • For freq type, enter 'z' for redshift or 'f' for frequency.
  • For galaxy classifications, enter one or more of the following separated by commas:
  • LBG (Lyman Break Galaxy)
  • MS (Main Sequence Galaxy)
  • SMG (Submillimeter Galaxy)
  • DSFG (Dusty Star-Forming Galaxy)
  • SB (Starburst)
  • AGN (Active Galactic Nucleus
  • QSO (Optically Bright AGN)
  • Quasar (Optical and Radio Bright AGN)
  • RQ-AGN (Radio-Quiet AGN)
  • RL-AGN (Radio-Loud AGN)
  • RG (Radio Galaxy)
  • BzK (BzK-selected Galaxy)

If your uncertainties are symmetric (positive and negative uncertainties are the same), it's okay to only fill in the column for the positive uncertainties and leave the 2nd column blank.
