Submitting Entries from a File

In order to upload multiple detections from a file, you must use a CSV file with correct format. Please download one of the template files below (for symmetric or assymetric uncertainties). Files that do not match the required format be automatically rejected. If your submission is rejected, please see the output warnings for possible corrections, and then resubmit. If you have further issues, please contact the database admin. All submissions will be manually approved before becoming publically accessible.

While entering values, please note that:

  • Coordinate System must be ICRS or J2000 only
  • For Lensing Flag, enter either Lensed, Unlensed or Unknown
  • For Freq Type, enter 'z' for Redshift or 'f' for Frequency.
  • For Galaxy Classifications, enter one or more of the following:
LBG (Lyman Break Galaxy), MS (Main Sequence Galaxy), SMB (Submillimeter Galaxy), DSFG (Dusty Star-Forming Galaxy), SB (Starburst), AGN (Contains a Known Active Galactic Nucleus), QSO (Optically Bright AGN), Quasar (Optical and Radio Bright AGN), Q-AGN (Radio-Quiet AGN), RL-AGN (Radio-Loud AGN), RG (Radio Galaxy), BZK (BZK-Selected Galaxy)
  • If your assymetries are symmetric (positive and negative uncertainties are the same), it's okay to only fill in the column for the positive uncertainties.
